# REQUIREMENTS
* You need python 2, sklearn and related libraries to run these scripts
* You also need a (postgres) database

# RUN

1. Make a `config.py` like `config.example.py` with your environment
2. Run `ingest.py`
3. run `etl.py`
4. run `predict.py`
